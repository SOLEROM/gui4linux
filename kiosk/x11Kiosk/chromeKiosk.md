# chromeKiosk

settings:

```
chromium-browser 
--kiosk

--window-size=1920,1080 --start-fullscreen --incognito --noerrdialogs --disable-translate --no-first-run --fast --fast-start --disable-infobars --disable-features=TranslateUI --disk-cache-dir=/dev/null  --password-store=basic



    --disable-pinch - disables the pinch-to-zoom functionality
    --overscroll-history-navigation=0 - disables the swipe-to-navigate functionality




```
